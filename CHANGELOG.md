# Changelog

All notable changes to this project will be documented in this file.

## [1.00] - 2025-12-08

### Added
- Initial release
- CSV import functionality for fightcard data
- Manual match entry with placeholder text
- Edit match dialog for modifying existing matches
- Remove match with confirmation dialog
- Sort matches by match number
- PDF generation using custom template
- Standalone Windows executable
- GUI with event name, file selection, and match management
- Scrollable match table for large fightcards

### Technical
- Modular code structure (core/, ui/ directories)
- PyInstaller packaging for Windows
- Resource path handling for bundled files
